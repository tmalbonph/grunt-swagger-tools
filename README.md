## Serenity Grunt module for Swagger validation

### Initial version

This is an initial public release

### How to install

``` install bower
npm install -g bower
```

``` install grunt
npm install -g grunt-cli
npm install grunt
```

``` install dependencies
npm install
```

### How to test

```
npm test
```
or
```
grunt test
```

### Test with JSON logs

```
npm run-script demo
```
or
```
grunt demo
```

### Test without JSON logs

```
npm run-script nolog
```
or
```
grunt nolog
```
